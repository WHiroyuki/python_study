github操作
  - ファイルの作成

    - ファイル作成
      ファイルを新規で作成

    - ファイル一時追加
      作成したファイルのディレクトリまでcd
      「git add xxx」を実行

    - ファイルコミット
      「git commit -m "add file xxx"」を実行

    - ファイルをリモートリポジトリへ登録
      「git push -u origin master」を実施

  - ファイルの削除

    - 削除コマンド
      「git rm -f xxx」を実施

    - ファイルをリモートリポジトリへ登録
      「git push -u origin master」を実施

全量更新
git add .
git commit -m "add"
